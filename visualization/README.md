# Apache Superset Light Mode (Docker)

This repository provides a **lightweight Docker setup** for Apache Superset with **automatic user creation**, using PostgresDB as the backend. 
Perfect for **development, testing, and local dashboards**.

---

## Features

- Runs Superset in **light mode**
- Automatic **user creation** on first start
- Single-command startup with `docker compose up -d`
- Customizable admin credentials via environment variables
- Persistent data using a Docker volume
